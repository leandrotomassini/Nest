# Teslo API

1. Clonar proyecto
2. ```yarn install```
3. Clonarl el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levatar la base de datos
```
docker-compose up -d
```
6. Ejecutar SEED
```
http://localhost:3000/api/seed
```
7. Levantar el modo de desarrollo: 
``` 
yarn start dev 
```
